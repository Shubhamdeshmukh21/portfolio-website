My Skills

HTML5

HTML is the foundation of any webpage, providing its structure and content. Think of HTML as the skeleton of a website, where each element, such as headings, paragraphs, images, and links, is defined with specific tags. HTML ensures the content is organized and readable by browsers

CSS3

CSS is responsible for the visual styling of a webpage. It controls colors, fonts, layouts, and overall aesthetics, transforming plain HTML into a visually appealing interface.

JavaScript

JavaScript is the programming language that brings functionality and interactivity to a webpage. It allows developers to create dynamic content, like forms that validate user input, interactive sliders, or real-time updates without refreshing the page..

React.js

React.js is a powerful JavaScript library used to build user interfaces, particularly for single-page applications (SPAs) where data changes over time. Developed by Facebook and released in 2013, React allows developers to create reusable UI components, manage state efficiently, and render views dynamically.

Figma

Figma is a powerful web-based design tool that allows users to create user interfaces, prototypes, and vector graphics. It’s widely used for collaborative design workflows due to its real-time collaboration features, which enable teams to work together on the same design file simultaneously.
